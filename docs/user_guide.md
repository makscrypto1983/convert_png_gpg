# Руководство пользователя для конвертера PNG в JPG

## Обзор

Конвертер PNG в JPG - это простое графическое приложение, которое позволяет
пакетно преобразовывать изображения в формат JPG с настраиваемыми параметрами качества.
Приложение поддерживает преобразование изображений в форматах PNG, WEBP, BMP и GIF.

## Системные требования

- Python 3.6 или выше
- Операционная система: Windows, macOS или Linux

## Установка

  1. Убедитесь, что Python установлен в вашей системе
  2. Загрузите или клонируйте исходный код приложения
  3. Установите необходимые зависимости:
     ```bash
     pip install -r requirements.txt
     ```
  4. Запустите приложение:
     ```bash
     python main.py
     ```

## Использование приложения

### Выбор входной директории

  1. Запустите приложение, выполнив `python main.py`
  2. Нажмите кнопку "Обзор" в разделе "Входная папка", чтобы выбрать директорию с изображениями (поддерживаются форматы PNG, WEBP, BMP, GIF)
  3. Выбранный путь к директории появится в текстовом поле

### Настройка выходной директории

  1. Нажмите кнопку "Обзор" в разделе "Настройки вывода"
  2. Выберите директорию, в которой вы хотите сохранить преобразованные JPG файлы
  3. Выбранный путь к директории появится в текстовом поле

### Регулировка качества

  1. Используйте поле со стрелками с надписью "Качество JPG (1-100)", чтобы установить качество вывода
  2. Более высокие значения (ближе к 100) приводят к лучшему качеству, но большим размерам файлов
  3. Более низкие значения (ближе к 1) приводят к меньшим размерам файлов, но более низкому качеству
  4. Значение качества по умолчанию - 95

### Настройка разрешения

  1. Используйте выпадающий список "Пресет разрешения" для быстрого выбора распространенных разрешений (Full HD, HD, 4K и др.)
  2. Или введите значения ширины и высоты вручную в соответствующие поля
  3. Используйте флажок "Сохранить соотношение сторон", чтобы сохранить пропорции исходного изображения при изменении размера

### Выбор темы

  1. Используйте выпадающий список "Тема" для выбора предпочтительной темы: системная, светлая или темная
  2. Приложение автоматически адаптируется под выбранную тему

### Преобразование изображений

  1. После выбора входной директории, настройки выходной директории, качества и разрешения:
  2. Нажмите кнопку "Преобразовать в JPG"
  3. Индикатор прогресса покажет прогресс преобразования
  4. Сообщения о статусе будут появляться в строке состояния внизу
  5. По завершении диалоговое окно покажет результаты преобразования

## Поддерживаемые форматы изображений

Приложение поддерживает преобразование следующих форматов изображений в JPG:
- PNG (Portable Network Graphics) - включая изображения с прозрачностью
- WEBP (Web Picture format) - популярный формат от Google
- BMP (Bitmap image file) - стандартный формат Windows
- GIF (Graphics Interchange Format) - примечание: для анимированных GIF будет преобразован только первый кадр

## Особенности

  - Пакетное преобразование нескольких файлов изображений из выбранной директории
 - Регулировка качества для выходных JPG файлов
  - Настройка разрешения с возможностью выбора пресетов
 - Сохранение исходного соотношения сторон изображения
 - Обработка файлов с прозрачностью (преобразование в белый фон)
  - Поддержка темного/светлого режима с определением системной темы
 - Отслеживание прогресса во время преобразования
  - Обработка ошибок для недействительных файлов

## Советы

  - Организуйте ваши файлы изображений в одной папке перед преобразованием для более легкого выбора файлов
 - Учитывайте соотношение качества и размера файла в зависимости от ваших потребностей
  - Используйте настройку выходной директории по умолчанию в config/settings.json, чтобы установить предпочтительное место
  - Используйте пресеты разрешения для быстрой настройки стандартных размеров изображений
