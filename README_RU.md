# Конвертер PNG в JPG

Простое GUI-приложение для пакетного преобразования изображений в формат JPG с настраиваемыми параметрами качества и разрешения. Приложение поддерживает несколько форматов изображений и предоставляет интуитивный интерфейс для преобразования изображений.

## Особенности функционала

- Графический интерфейс для простого преобразования изображений
- Пакетное преобразование нескольких изображений из выбранной директории
- Настройка качества выходных изображений (1-100)
- Настройка разрешения с контролем ширины и высоты
- Опция сохранения соотношения сторон при изменении размера
- Несколько пресетов разрешения (Full HD, HD, 4K и др.)
- Поддержка темного/светлого режима с определением системной темы
- Настраиваемый каталог вывода
- Индикатор прогресса для отслеживания процесса конвертации
- Поддержка обработки прозрачности (изображения с альфа-каналами)
- Поддержка нескольких входных форматов: PNG, WEBP, BMP и GIF

## Требования к системе

- Python 3.6 или выше
- tkinter (обычно включена в Python)
- Pillow для обработки изображений
- ttkthemes для расширенного оформления (опционально)

## Инструкции по установке

  1. Клонируйте этот репозиторий или загрузите исходный код
  2. Установите необходимые зависимости:
     ```bash
     pip install -r requirements.txt
     ```
  3. Запустите приложение:
     ```bash
     python main.py
     ```

## Инструкции по использованию

  1. Запустите приложение, выполнив `python main.py`
  2. Выберите входную директорию с изображениями с помощью кнопки "Обзор" в разделе "Входная папка"
  3. Установите каталог вывода, куда будут сохраняться преобразованные изображения, с помощью кнопки "Обзор" в разделе "Настройки вывода"
  4. Настройте параметры качества (1-100) при необходимости - более высокие значения означают лучшее качество, но больший размер файла
  5. При необходимости выберите пресет разрешения или установите пользовательские значения ширины и высоты для выходных изображений
  6. Используйте флажок "Сохранить соотношение сторон" для сохранения оригинальных пропорций при изменении размера
  7. Выберите тему (системная, светлая или темная) из выпадающего списка темы
  8. Нажмите "Преобразовать в JPG", чтобы начать процесс преобразования
 9. Отслеживайте индикатор прогресса и сообщения о статусе во время конвертации
  10. Найдите преобразованные JPG изображения в указанном каталоге вывода

## Примеры использования

 1. **Базовое преобразование**: Выберите директорию с PNG, WEBP, BMP или GIF файлами, выберите каталог вывода, нажмите "Преобразовать в JPG"
  2. **Настройка качества**: Установите качество 85 для меньшего размера файла с хорошим качеством
  3. **Изменение разрешения**: Выберите пресет "1920x1080 (Full HD)" для изменения размера всех изображений до Full HD разрешения
  4. **Пакетное преобразование**: Преобразуйте все поддерживаемые изображения в выбранной директории за один раз
 5. **Конвертация форматов**: Преобразуйте изображения из любого поддерживаемого формата (PNG, WEBP, BMP, GIF) в JPG

## Описание интерфейса

Приложение имеет чистый и интуитивно понятный интерфейс со следующими компонентами:
- Панель выбора входной директории
- Выбор каталога вывода с кнопкой обзора
- Поле управления качеством (1-100)
- Выпадающий список пресетов разрешения (Full HD, HD, 4K и др.)
- Элементы управления разрешением (ширина, высота, сохранение соотношения сторон)
- Выбор темы (системная, светлая, темная)
- Индикатор прогресса, показывающий статус преобразования
- Статусная строка с обновлениями в реальном времени
- Кнопка преобразования для запуска процесса конвертации

## Конфигурация

Приложение может быть настроено с помощью файла `config/settings.json`. Вы можете изменить настройки по умолчанию, такие как:
- Каталог вывода по умолчанию
- Качество изображения по умолчанию
- Последняя использованная входная директория
- Последние настройки разрешения
- Предпочтение темы (системная, светлая, темная)
- Настройка сохранения соотношения сторон

## Поддерживаемые входные форматы

Приложение поддерживает преобразование из следующих форматов изображений в JPG:
- PNG (Portable Network Graphics)
- WEBP (Web Picture format)
- BMP (Bitmap image file)
- GIF (Graphics Interchange Format) - Примечание: будет преобразован только первый кадр анимированных GIF
